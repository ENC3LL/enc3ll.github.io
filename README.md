## üá¨üáß English Description

**Asakaro** is a beautiful and functional Markdown editor designed to be your personal **knowledge vault**. It features a modern **Glassmorphism** UI and stands out with unique capabilities like integrated PDF viewing alongside standard Markdown editing.

The project was built entirely from scratch using **TypeScript**, **HTML**, and **CSS**. No heavy frameworks or libraries were used‚Äîeven the Markdown parsing and syntax highlighting engine was custom-written to ensure maximum control and performance.

### üåü Key Features
* **Glassmorphism Design:** A sleek, modern aesthetic that looks great on any screen.
* **Unified Viewer:** Open and manage both `.md` files and `.pdf` documents in one place.
* **Persistent Storage:** Files are saved locally, so your knowledge base is always ready when you return.
* **Custom Engine:** The core logic, including text rendering and styling, is hand-crafted in pure TypeScript.
