ГЛАВА 1. КОМБИНАТОРИКА
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


────────────────────────────────────────────────────────────────────────────
1.1  ПЕРЕСТАНОВКИ БЕЗ ПОВТОРЕНИЙ
────────────────────────────────────────────────────────────────────────────

ФОРМУЛА:
    P(n) = n!
    n! = 1 · 2 · 3 · ... · n
    По определению: 0! = 1

Применяется когда все n элементов различны и нужно подсчитать число
способов их упорядочить. Порядок важен.

▸ ТИПОВЫЕ ЗАДАЧИ:

  Задача 1.1.1
  Условие: Сколькими способами можно расставить 5 человек на скамейке?
  Решение: P(5) = 5! = 120
  Ответ: 120 способов.

  Задача 1.1.2
  Условие: Сколько различных слов из букв слова ЗАМОК?
  Решение: P(5) = 5! = 120
  Ответ: 120 вариантов.

  Задача 1.1.3
  Условие: Из 8 участников — 3 призовых места (1-е, 2-е, 3-е). Сколько
           вариантов?
  Решение: Это размещение A(8,3) = 8·7·6 = 336  (см. раздел 1.3)

★ СЛОЖНАЯ ЗАДАЧА:

  Задача 1.1.★
  Условие: На 5 карточках буквы Б, Е, Р, С, Т. Карточки перемешали.
           P(при раскладке получится БРЕСТ)?
  Решение:
    Всего перестановок: n = 5! = 120
    Благоприятных: m = 1
    P = 1/120 ≈ 0,0083


────────────────────────────────────────────────────────────────────────────
1.2  ПЕРЕСТАНОВКИ С ПОВТОРЕНИЯМИ
────────────────────────────────────────────────────────────────────────────

ФОРМУЛА:
    P(n; n1, n2, ..., nk) = n! / (n1! · n2! · ... · nk!)
    где n1 + n2 + ... + nk = n

Используется когда среди n элементов есть одинаковые:
n1 штук первого вида, n2 штук второго и т.д.

▸ ТИПОВЫЕ ЗАДАЧИ:

  Задача 1.2.1
  Условие: Перестановки из букв слова РОТОР?
  Решение: Р — 2, О — 2, Т — 1. n = 5.
           P = 5! / (2!·2!·1!) = 120/4 = 30
  Ответ: 30.

  Задача 1.2.2
  Условие: Перестановки из букв слова КОЛОКОЛ?
  Решение: К — 2, О — 3, Л — 2. n = 7.
           P = 7! / (2!·3!·2!) = 5040/24 = 210
  Ответ: 210.

  Задача 1.2.3
  Условие: 6-значные числа из цифр 1,1,1,2,2,2?
  Решение: P = 6! / (3!·3!) = 720/36 = 20
  Ответ: 20.

★ СЛОЖНАЯ ЗАДАЧА:

  Задача 1.2.★
  Условие: На 6 карточках буквы ТАЛАНТ. Перемешали.
           P(получить слово ТАЛАНТ)?
  Решение:
    Т — 2, А — 2, Л — 1, Н — 1.
    Всего: P = 6!/(2!·2!·1!·1!) = 720/4 = 180
    m = 1

────────────────────────────────────────────────────────────────────────────
1.3  РАЗМЕЩЕНИЯ
────────────────────────────────────────────────────────────────────────────

ФОРМУЛА (без повторений):
    A(n,m) = n! / (n-m)! = n·(n-1)·...·(n-m+1)

Упорядоченный выбор m элементов из n различных. Порядок важен!

ФОРМУЛА (с повторениями):
    (A(n,m))_повт = n^m

Когда каждый элемент может повторяться.

  Решение: A(10,3) = 10·9·8 = 720
  Ответ: 720 способов.

  Задача 1.3.2
  Условие: 4-значные коды из 0–9 без повторений.
  Решение: A(10,4) = 10·9·8·7 = 5040

  Задача 1.3.3
  Условие: 4-значные коды из 0–9 с повторениями.
  Решение: (A(10,4))_повт = 10^4 = 10000

★ СЛОЖНАЯ ЗАДАЧА:

  Условие: Абонент забыл 3 последних цифры номера, помнит только что

────────────────────────────────────────────────────────────────────────────
1.4  СОЧЕТАНИЯ
────────────────────────────────────────────────────────────────────────────

ФОРМУЛА (без повторений):
    C(n,m) = n! / (m! · (n-m)!)

Неупорядоченный выбор m элементов из n. Порядок НЕ важен.
Свойства: C(n,m) = C(n, n-m);  C(n,0) = 1

ФОРМУЛА (с повторениями):
    (C(n,m))_повт = C(n+m-1, m)

▸ ТИПОВЫЕ ЗАДАЧИ:

  Задача 1.4.1
  Условие: 3 человека на 3 одинаковые должности из 10 кандидатов.
  Решение: C(10,3) = 10!/(3!·7!) = 120
  Ответ: 120 способов.

  Задача 1.4.2
  Условие: В ящике 4 синих и 5 красных шаров. Вынимают 2. P(разного цвета)?
  Решение: Всего: C(9,2) = 36
           Благоприятных: C(4,1)·C(5,1) = 4·5 = 20
           P = 20/36 = 5/9 ≈ 0,556

  Задача 1.4.3
  Условие: В партии 10 деталей, 7 стандартных. Берут 6. P(4 стандартные)?
  Решение: C(10,6) = 210
           C(7,4)·C(3,2) = 35·3 = 105
           P = 105/210 = 0,5

ГЛАВА 2. КЛАССИЧЕСКОЕ ОПРЕДЕЛЕНИЕ ВЕРОЯТНОСТИ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ФОРМУЛА:
    P(A) = m / n

    m — число элементарных исходов, благоприятствующих A
    n — общее число равновозможных исходов

Свойства:
    0 <= P(A) <= 1
    P(Omega) = 1   (достоверное событие)
    P(пустое) = 0  (невозможное событие)

ГЕОМЕТРИЧЕСКАЯ ВЕРОЯТНОСТЬ:
    P(A) = mu(g) / mu(G)
    mu — мера (длина, площадь, объём).
    Применяется когда пространство исходов непрерывно.

  Решение: n = 30.  m = 6 (числа 5, 10, 15, 20, 25, 30)
           P = 6/30 = 0,2

ГЛАВА 3. ФОРМУЛА ПОЛНОЙ ВЕРОЯТНОСТИ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ФОРМУЛА:
    P(A) = P(H1)·P(A/H1) + P(H2)·P(A/H2) + ... + P(Hn)·P(A/Hn)
    P(A) = sum_i [ P(Hi) · P(A/Hi) ]

    H1, H2, ..., Hn — ПОЛНАЯ ГРУППА ГИПОТЕЗ:
      - попарно несовместны
      - sum P(Hi) = 1
    P(A/Hi) — условная вероятность A при условии что произошло Hi

Алгоритм:
  1) Выделить все гипотезы H1...Hn (sum P(Hi) = 1)
  2) Для каждой гипотезы записать P(A/Hi)
  3) Подставить в формулу

▸ ТИПОВЫЕ ЗАДАЧИ:

  Условие: Торпеда. P(в нос)=0,3; P(в ср.)=0,4; P(в корму)=0,3.

           Найти P(потопления).
  Решение: P(A) = 0,3·0,5 + 0,4·0,9 + 0,3·0,6
                = 0,15 + 0,36 + 0,18 = 0,69

           машина 2 — 25% (брак 1%), машина 3 — 45% (брак 3%).
           P(болт бракованный)?
  Решение: P(A) = 0,30·0,02 + 0,25·0,01 + 0,45·0,03
                = 0,006 + 0,0025 + 0,0135 = 0,022

  Задача 3.3
  Условие: Группа: 5 отличников, 10 хорошистов, 6 слабых (21 чел.).

ГЛАВА 4. ФОРМУЛА БАЙЕСА
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ФОРМУЛА:
    P(Hk/A) = P(Hk) · P(A/Hk) / P(A),   k = 1, 2, ..., n

    P(A) — из формулы полной вероятности
    P(Hk/A) — АПОСТЕРИОРНАЯ вероятность (после наступления A)

Разница:
  Полная вероятность: "каков будет результат?"
  Байес:              "из какого источника пришёл результат?"

Свойство: sum P(Hk/A) = 1

▸ ТИПОВЫЕ ЗАДАЧИ:

  Задача 4.1
  Условие: Условие задачи 3.2. Болт бракованный.
           P(он изготовлен машиной 3)?
  Решение: P(A) = 0,022
           P(H3/A) = 0,45·0,03 / 0,022 = 0,0135/0,022 ≈ 0,614

  Задача 4.2
  Условие: Склад: фабрика 1 — 20% (брак 3%), фабрика 2 — 46% (брак 2%),
           фабрика 3 — 34% (брак 1%). Изделие нестандартное.
           P(оно с фабрики 1)?
  Решение: P(A) = 0,20·0,03 + 0,46·0,02 + 0,34·0,01 = 0,0186
           P(H1/A) = 0,20·0,03 / 0,0186 = 0,006/0,0186 ≈ 0,322

  Задача 4.3
  Условие: Условие задачи 3.3. Студент получил хорошую/отл. оценку.
           P(это был отличник)?
  Решение: P(A) = 17/21
           P(H1/A) = (5/21)·1 / (17/21) = 5/17 ≈ 0,294

★ СЛОЖНАЯ ЗАДАЧА:

  Задача 4.★
  Условие: Телеграф: 3/5 — точки, 2/5 — тире. Помехи: 2/5 точек
           превращаются в тире, 1/3 тире — в точки.
           Принята "точка". P(передавалась точка)?
  Решение:
    H1 = передавалась точка: P(H1) = 3/5
    H2 = передавалось тире:  P(H2) = 2/5
    P(A/H1) = 1 - 2/5 = 3/5  (точка не искажена)
    P(A/H2) = 1/3             (тире → точка)
    P(A) = (3/5)·(3/5) + (2/5)·(1/3) = 9/25 + 2/15 = 37/75
    P(H1/A) = (9/25) / (37/75) = (9/25)·(75/37) = 27/37 ≈ 0,730

ГЛАВА 5. ФОРМУЛА БЕРНУЛЛИ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ОСНОВНАЯ ФОРМУЛА:
    Pn(m) = C(n,m) · p^m · q^(n-m)

    n — число независимых испытаний
    p — P(A) в одном опыте
    q = 1 - p
    m — искомое число появлений A

Условие применения: испытания независимы, p = const.

СЛЕДСТВИЯ:

  1. Хотя бы раз (m >= 1):
       Pn(m >= 1) = 1 - q^n

  2. Не менее k раз (k > n/2):
       Pn(m >= k) = sum_{m=k}^{n} Pn(m)

  3. Не более k раз:
       Pn(m <= k) = sum_{m=0}^{k} Pn(m)

  4. Наивероятнейшее число m*:
       np - q  <=  m*  <=  np + p
     (если правая граница — целое, m* принимает два значения)

  5. Минимальное число опытов для P(хотя бы раз) >= P_зад:
       n >= lg(1 - P_зад) / lg(q)

           n >= lg(0,1)/lg(0,7) = -1/-0,1549 ≈ 6,46  =>  n = 7

    16 <= 0,7n + 0,7  =>  n >= 21,86
    Ответ: n = 22 или n = 23

ГЛАВА 6. ФОРМУЛА ПУАССОНА
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ФОРМУЛА:
    Pn(m) ≈ a^m / m! · e^(-a),   где  a = n · p

Условие применения:
  - n велико (n -> inf)
  - p мало   (p -> 0)
  - a = np = const  (a <= 10)
  Чем точнее выполнены условия, тем точнее результат.

Константа: e ≈ 2,71828

Таблица e^(-a):
  a  |  1      2      3      4      5      6      7      8
  ---|--------------------------------------------------------
  e⁻ᵃ| 0,368  0,135  0,050  0,018  0,007  0,002  0,001  0,000

▸ ТИПОВЫЕ ЗАДАЧИ:

  Задача 6.1
  Условие: Завод отправляет 3000 изделий. P(повреждения) = 0,001.
           P(ровно 5 повреждённых)?
  Решение: a = 3000·0,001 = 3;  m = 5
           P ≈ 3^5/5! · e^(-3) = 243/120 · 0,050 ≈ 0,101

  Задача 6.2
  Условие: 1000 абонентов. P(звонок за 1 мин) = 0,003.
           P(ровно 2 звонка за 1 мин)?
  Решение: a = 1000·0,003 = 3;  m = 2
           P ≈ 3^2/2! · e^(-3) = 9/2 · 0,050 = 0,225

  Задача 6.3
  Условие: n=100, p=0,01. P(не менее 3 отказов)?
  Решение: a = 1;
           P(0) = e^(-1) = 0,368
           P(1) = 1·e^(-1) = 0,368
           P(2) = 0,5·e^(-1) = 0,184
           P(m>=3) = 1 - 0,368 - 0,368 - 0,184 = 0,080

★ СЛОЖНАЯ ЗАДАЧА:

  Задача 6.★
  Условие: Аппаратура за 10 000 ч выходит из строя 10 раз.
           Операция — 500 ч.
           P(выйдет ровно 1 раз) и P(хотя бы 1 раз).
  Решение:
    p = 10/10000 = 0,001 за 1 час
    За 500 ч: a = 500·0,001 = 0,5
    P(1) ≈ 0,5^1/1! · e^(-0,5) = 0,5·0,6065 ≈ 0,303
    P(m>=1) = 1 - P(0) = 1 - e^(-0,5) = 1 - 0,6065 ≈ 0,394

ИТОГОВАЯ ТАБЛИЦА ВСЕХ ФОРМУЛ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  #  | Название                   | Формула                              | Когда применять
  ---|----------------------------|------------------------------------- |-----------------------------
  1  | Перестановки               | P(n) = n!                            | Все n различны
  2  | Перест. с повторениями     | P = n! / n1!·n2!·...·nk!             | Есть одинаковые элементы
  3  | Размещения                 | A(n,m) = n!/(n-m)!                   | Порядок важен, без повтора
  4  | Размещения с повт.         | (A(n,m))_повт = n^m                  | Порядок важен, с повтором
  5  | Сочетания                  | C(n,m) = n!/m!(n-m)!                 | Порядок не важен
  6  | Класс. вероятность         | P(A) = m/n                           | Равновозможные исходы
  7  | Полная вероятность         | P(A) = sum P(Hi)·P(A/Hi)             | Полная группа гипотез
  8  | Формула Байеса             | P(Hk/A) = P(Hk)·P(A/Hk)/P(A)        | Известен результат → причина
  9  | Формула Бернулли           | Pn(m) = C(n,m)·p^m·q^(n-m)          | n незав. испытаний, p=const
  10 | Хотя бы раз                | Pn(m>=1) = 1 - q^n                   | Следствие Бернулли
  11 | Наивероятнейшее m*         | np-q <= m* <= np+p                   | Ищем наиболее вероятное m
  12 | Мин. кол-во опытов         | n >= lg(1-P_зад)/lg(q)               | Гарантировать P(A) >= P_зад
  13 | Формула Пуассона           | Pn(m) ≈ a^m/m! · e^(-a), a=np       | n>>1, p<<1, np<=10

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
БЫСТРАЯ ШПАРГАЛКА: КАК ВЫБРАТЬ ФОРМУЛУ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Задача на подсчёт числа вариантов?
    ├─ Порядок важен?
    │    ├─ Да → РАЗМЕЩЕНИЕ  A(n,m) = n!/(n-m)!
    │    └─ Нет → СОЧЕТАНИЕ  C(n,m) = n!/m!(n-m)!
    └─ Все n элементов задействованы?
         ├─ Да, все различны → ПЕРЕСТАНОВКА  P(n) = n!
         └─ Да, есть одинаковые → ПЕРЕСТ. С ПОВТ.  P = n!/n1!·...·nk!

  Задача на вычисление вероятности?
    ├─ Есть гипотезы, ищем P(результата) → ПОЛНАЯ ВЕРОЯТНОСТЬ
    ├─ Известен результат, ищем P(причины) → БАЙЕС
    ├─ n одинаковых независимых опытов, p = const?
    │    ├─ n умеренное → БЕРНУЛЛИ
    │    └─ n велико, p мало, np<=10 → ПУАССОН
    └─ Конечное равновозможное пространство → КЛАССИЧЕСКАЯ P = m/n
